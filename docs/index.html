Reach

Leap Motion OSC Mapper

A minimal interface that provides immediate access to the core settings whilst giving visual feedback of the hand detection.

An intuitive use of the OSC format makes the data easy to parse.

The Open Source nature of the software makes it easy to manipulate. Be part of the community and make sure to request features by posting an issue on GitHub!
